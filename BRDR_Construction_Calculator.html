BRDR Construction Calculator

Instructions:
• Upgrade Time (Days): Enter the upgrade time shown in-game before buffs.
• Total Construction Buff (%): Add up all your construction buffs including Secretary of Development if active.
• Daily Free Speed-Up Time (Hours per Day): Enter how many free speed-up hours you earn daily from buffs.
• Number of Days You Want to Calculate: Usually the actual upgrade time after buffs (or how long you plan to run the upgrade).
• Number of Helps Received: Enter how many times you expect to receive help from alliance members.
• Help Time per Help (Minutes): Enter the help duration per help. If you don’t know, 7.5 minutes is a common default.
